# Data-Analysis-Course-Project
A work-in-progress Power BI project built to practice and explore its core features and capabilities.


<img width="1356" height="685" alt="image" src="https://github.com/user-attachments/assets/b68235a7-3ddc-414f-ab53-8cd72c621295" />
Developed key financial measures in Power BI to analyze company performance, including yearly gross margin profit, prior-year profit comparison, and year-over-year differences in both absolute values and percentages. The report allows dynamic year selection through interactive filters.
